# Glossaire

La terminologie utilisée dans ECI.

### Active Inference
Un cadre théorique où les agents agissent pour minimiser leur surprise (énergie libre) par rapport à leurs modèles internes.

### HGF (Hierarchical Gaussian Filter)
Le modèle utilisé pour simuler la manière dont les électeurs mettent à jour leurs croyances face à de nouvelles informations (les politiques des candidats).

### Tonic Volatility
Un paramètre définissant la volatilité de base d'un agent. Un agent avec une volatilité élevée changera d'avis plus facilement.
